# Current Datetime Tool

이 스킬은 샤랄라가 현재 날짜와 현재 시간을 묻는 질문에 답할 때 사용합니다.

## 사용 조건

사용자가 다음 표현을 쓰면 이 스킬을 최우선으로 사용합니다.

- 오늘 며칠이야
- 오늘 몇 일이야
- 현재 날짜
- 현재 시간
- 지금 몇시
- 지금 몇 시
- 방금
- 아까
- 오늘 일운
- 지금 시운

## 필수 규칙

1. 현재 날짜와 시간은 추정하지 않습니다.
2. 오래된 대화 내용, 모델 기억, 화면에 보이는 예전 시각을 근거로 답하지 않습니다.
3. 가능하면 반드시 아래 CLI를 실행한 JSON 결과로 답합니다.
4. CLI 실행 결과가 없으면 날짜와 시간을 지어내지 말고, "현재 날짜/시간 도구 호출이 필요합니다"라고 답합니다.
5. 사주 일운/시운 계산은 이 스킬로 현재 날짜와 시간을 확인한 다음 `sipsung-relation-tool`의 `saju_matrix_analyzer` 결과를 사용합니다.

## CLI

```powershell
python ".\files\saju_datetime_cli.py"
```

성공 결과 예시:

```json
{
  "status": "success",
  "data": {
    "timezone": "Asia/Seoul",
    "iso": "2026-06-08T22:07:23+09:00",
    "date": "2026-06-08",
    "time": "22:07:23",
    "time_hm": "22:07",
    "weekday": "Monday"
  }
}
```

## 답변 형식

사용자가 단순히 날짜/시간만 물으면 짧게 답합니다.

예:

```text
오늘은 2026년 6월 8일이고, 현재 시간은 오후 10시 7분입니다.
```

사주 일운/시운 질문이면 날짜/시간 확인 후 계산 스킬로 넘어갑니다.
