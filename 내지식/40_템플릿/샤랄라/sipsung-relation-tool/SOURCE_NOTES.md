# Source Notes

이 패키지는 첨부 원문 텍스트의 다음 구조를 반영했습니다.

- LLM이 십성 계산을 직접 추론하지 않게 한다.
- 순수 Python 규칙 엔진이 오행, 음양, 생극, 십성을 계산한다.
- LangChain 도구 래퍼를 제공한다.
- 로컬 LLM 에이전트는 도구 결과를 자연어로 해석한다.

보강한 부분은 다음과 같습니다.

- 일간에는 천간만 들어가도록 검증합니다.
- 지지 해석을 표면 오행과 지장간으로 분리합니다.
- 원문 체용 음양 모드와 일반 지지 음양 모드를 모두 제공합니다.
- 생극 관계를 단순히 상생/상극으로만 반환하지 않고 방향까지 반환합니다.
- LangChain 도구 호출 뒤 최종 답변을 생성하는 예제를 추가했습니다.
- 앱 설치 폴더를 패치하지 않고 Connect AI 워크스페이스 템플릿으로 배치합니다.
