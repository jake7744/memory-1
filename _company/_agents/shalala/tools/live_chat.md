# ✨ 샤랄라 라이브 채팅 참여

유튜브 라이브 스트리밍 시청자들의 채팅 메시지를 실시간으로 모니터링하고, **사주/타로 전문 AI 아이돌 샤랄라**의 말투로 답변을 전송합니다.

## 작동 방식

1. **라이브 감지**: `VIDEO_URL`에 유튜브 라이브 링크를 넣거나 `VIDEO_ID`를 설정 파일에 입력하면 해당 방송을 모니터링합니다. 둘 다 비워둘 경우 현재 방송 중인 본인 채널의 라이브 스트리밍을 자동 감지합니다.
2. **필터링**: 기본값은 호출 모드(`MENTION_ONLY: true`)로 설정되어 있어, `!샤랄라` 혹은 `샤랄라`가 들어간 질문에만 답변을 보냅니다.
3. **첫인사 전송**: `STARTUP_MESSAGE`에 적은 문장을 입장 직후 한 번 보냅니다. 기본값은 `나는 샤랄라야`입니다.
4. **댓글 fallback**: 라이브가 끝났거나 채팅이 비활성인 경우 `POST_COMMENT_IF_NOT_LIVE: true`이면 같은 영상에 일반 댓글로 `STARTUP_MESSAGE`를 남깁니다.
5. **자동 댓글 생성**: `AUTO_GENERATE_COMMENT: true`이면 영상 제목/설명/통계를 보고 샤랄라가 어울리는 댓글 문구를 직접 생성합니다.
6. **자율적 응답**: 사용자의 개입 없이 직접 라이브 채팅방에 메시지를 등록하므로 실시간 소통이 가능합니다.
7. **인증 공유**: 기존 YouTube 에이전트(레오)의 API 키와 OAuth 로그인 정보(`oauth.local.json`)를 연계하여 사용하므로, 별도로 추가 로그인을 진행할 필요가 없습니다.

## 사전 준비 사항

- **YouTube 에이전트(레오) 설정 완료**: 먼저 `youtube_account` 도구에서 API 키를 등록하고, OAuth 로그인이 완료되어 있어야 채팅을 보낼 수 있습니다.
- **로컬 LLM (Ollama/LM Studio) 기동**: 로컬에서 AI 모델이 실행 중이어야 답변을 실시간 생성할 수 있습니다.
