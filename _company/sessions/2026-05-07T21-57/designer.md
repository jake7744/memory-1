### 📜 [프로젝트명: 더바이브] 통합 기술 사양서 (Master Technical Specification) v1.0
*목표: 스크립트 $\rightarrow$ 감정적 아크 $\rightarrow$ 시각 API 규격 적용 (Deep Indigo $\to$ Cream/Gold)*
*작성 근거: V4.0 표준 비주얼 템플릿 API 규격서, Deep Indigo-Cream-Gold 팔레트 시스템*

**[⚠️ 전제 조건 및 지침]**
1.  **Time Code 기준:** 모든 사양은 시작 시간($\text{Start Time}$)과 끝 시간($\text{End Time}$)을 포함하며, 오차 범위는 $\pm 0.2$초로 간주합니다.
2.  **색상 적용 우선순위:** 기본 배경 및 분위기는 Deep Indigo ($\#1A1E3D$)를 유지하고, 핵심 메시지나 깨달음이 발생하는 지점(Climax)에서만 Cream ($\#F4EBD0$) 또는 Gold ($\#B8860B$) 악센트를 사용합니다.
3.  **전환 효과:** 모듈 간 전환은 기본적으로 **Wipe Transition (12프레임, 0.2초)**을 적용하되, 감정적 아크가 급격히 변화하는 지점에서는 Dissolve(0.8s)를 활용합니다.
