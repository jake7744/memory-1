### Ⅰ. 브랜드 핵심 정의 (Visual Identity Core Rules)

**1. 컬러 팔레트 (Color Palette)**
| 이름 | 용도 | HEX 코드 | 비고 |
| :--- | :--- | :--- | :--- |
| **Deep Indigo** | 배경, 불안/미지 영역 | `#2B3A65` | 메인 배경색. 깊이와 사색을 상징. |
| **Cream Gold** | 깨달음, 하이라이트, CTA | `#F7D48E` | 긍정적 에너지를 상징하는 악센트 컬러. |
| **Accent Gold** | 강조 애니메이션 (골드 플로우) | `#FFC312` | Gold의 역동적인 움직임에 사용되는 주황빛 골드. |
| **Neutral Cream** | 본문 텍스트, UI 배경 | `#F5EBE0` | 가독성을 높이는 부드러운 흰색 대체재. |

**2. 타이포그래피 계층 구조 (Typography Hierarchy)**
*   **헤드라인 (H1/Title):** Noto Sans KR Bold / 48pt 이상. **[규칙]** 반드시 Cream Gold 또는 Accent Gold로 처리하여 시선 집중 유도.
*   **서브 헤드라인 (H2/Section Title):** Noto Sans KR SemiBold / 24pt. **[규칙]** Deep Indigo 배경 위에 Neutral Cream으로 배치.
*   **본문 텍스트 (Body Text):** Pretendard Regular / 16~20pt. **[규칙]** 최소 대비(Contrast)를 확보하며, 깊은 사유를 유도하는 간격감 유지.

### Ⅱ. UI/UX 플로우 및 인터랙션 원칙 (System Interaction Guidelines)

자동화 시스템의 경험은 단순한 '로딩'이 아닌 **'무의식의 작동 과정(The Process of Uncovering)'**을 시각화해야 합니다.

| 영역 | 목표 UX/UI 컨셉 | 구체적 디자인 규칙 및 좌표 지침 |
| :--- | :--- | :--- |
| **진입 (Entry)** | *깊은 몰입 유도.* 시스템 시작 전, 배경에 은은하게 움직이는 별자리 또는 노드맵의 '흐름' 애니메이션을 3초간 노출. | **배경:** Deep Indigo (opacity 0.9). **요소:** 미세한 골드(HEX #FFC312) 점들이 느린 속도로 연결되며 흐르는 모션 그래프 오버레이 적용. |
| **처리 중 (Processing)** | *기다림의 지루함 해소.* 단순 로딩 바 대신, '무의식이 데이터를 재구성하는' 과정을 시각화해야 함. | **컨테이너:** 중앙에 투명한 유리 질감(Glassmorphism) 박스를 배치하고, 내부에서 Indigo $\rightarrow$ Gold로 점진적인 색상 변화가 일어나는 **노드맵 애니메이션**을 루프 재생 (2초 주기). 로딩 텍스트는 "Connecting the Unconscious..."와 같이 전문적이고 사색적인 문구 사용. |
| **결과 노출 (Aha Moment)** | *강렬한 카타르시스.* 최종 결과물을 보여주는 순간은 빛이 터져 나오는 듯한 극적 전환을 제공해야 함. | **전환:** Deep Indigo에서 Cream Gold로의 급격하고 부드러운 색상 반전을 사용. 모든 UI 요소가 잠깐 동안 골드빛으로 오버레이 되었다가 정착하는 '플래시 효과'를 주입. CTA 영역은 이 순간의 중심에 배치되어야 함. |
| **오류 처리 (Failure State)** | *신뢰성 유지.* 오류 발생 시 당황스럽지 않게, 문제 해결 과정 자체를 안내함. | **디자인:** Deep Indigo 배경에 금색 악센트가 포함된 '진단 리포트' 느낌의 팝업창을 사용. 경고 아이콘은 일반적인 삼각형이 아닌, 미지의 에너지가 응축되는 듯한 육각형 형태를 채택하고, 해결책(Fix) 섹션에서 Cream Gold로 전환되도록 유도함. |

### Ⅲ. 반복 비주얼 요소 디자인 원칙 (Reusable Motion & Asset Rules)

**1. 트랜지션 규칙: Dissolve Gradient Transition (필수)**
*   **규칙:** 모든 장면 전환(Scene Transition)은 절대 하드컷(Hard Cut)을 사용하지 않습니다.
*   **구현 방식:** Deep Indigo 배경색에서 Cream Gold가 점진적으로 침투하며 서로를 녹여내는 'Dissolve Gradient' 효과를 적용합니다. (트랜지션 시간: 0.8초 ~ 1.2초).
*   **기술 스펙:** 배경 레이어에 `linear-gradient` 기반의 모션을 적용하고, 전환 지점에 맞춰 투명도(Opacity)가 점진적으로 변화하도록 설계해야 합니다.

**2. 핵심 강조 효과: Golden Pulse Glow (Golden Highlight)**
*   **규칙:** 사용자가 주목하거나 시스템이 특정 개념을 '깨달음'으로 정의하는 순간에만 발동합니다.
*   **구현 방식:** 해당 텍스트/요소 주변에 크기가 작고(radius: 2px) 밝은 Accent Gold 색상의 빛나는 파동(Pulse) 효과를 짧게 (0.1초 ~ 0.3초) 깜빡입니다. 이는 시청자의 무의식적인 집중을 유도하는 핵심 장치입니다.

**3. 로딩/데이터 처리 애니메이션: Flowing Node Map**
*   **규칙:** 시스템이 복잡한 데이터를 분석하고 연결하는 과정을 보여줍니다.
*   **구현 방식:** 3D 공간처럼 보이는 Deep Indigo 배경에 수백 개의 작은 노드(점)가 무작위로 생성되었다가, 데이터가 처리됨에 따라 특정 키워드를 중심으로 골드 플로우(`HEX #FFC312`)를 따라 연결되는 모션 그래프를 반복합니다. (속도: 느리고 유기적이어야 함.)
