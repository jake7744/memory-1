## 📘 AI 윤리 콘텐츠: T&M 가이드 업데이트 브리프 (V2.1)

새로운 테마(AI 윤리)를 반영하여, 기존의 감정적 흐름($\text{Deep Indigo} \rightarrow \text{Cream Gold}$)을 유지하되, 시각적 언어를 **'추상적인 시스템/데이터 구조'**에 초점을 맞춰 업데이트합니다. 이 브리프는 개발자가 코딩할 수 있는 JSON SSoT v2.0의 *비주얼 에셋 정의*와 *변형 규칙*에 최우선으로 반영되어야 합니다.

### 1. 🎨 비주얼 컨셉 및 목표 (Thematic Core)

| 요소 | 설명 | 적용 원칙 |
| :--- | :--- | :--- |
| **테마** | Digital Subconscious / Connection & Resolution | 인간의 무의식적 불안($\text{Indigo}$)이 데이터 시스템을 거쳐 논리적인 깨달음($\text{Gold}$)으로 해소되는 과정을 시각화. |
| **핵심 메타포** | 연결된 노드(Node), 흐르는 데이터 스트림(Data Stream), 알고리즘 패턴(Pattern) | 단순한 배경색 변화가 아닌, *구조*와 *논리*의 움직임을 통해 감정적 아크를 표현합니다. |
| **톤앤매너** | 깊고 신비로우며, 동시에 고도로 정교하고 지적인 느낌 (Sophisticated & Profound). |

### 2. ✨ 컬러 시스템 업데이트 및 데이터 변형 규칙

기존 색상 코드를 유지하되, '깊이'와 '흐름'을 표현하기 위해 **글로시/패턴 레이어**를 추가합니다.

#### A. 주조색 팔레트 (Primary Palette)

| 이름 | HEX 코드 | 역할 및 사용 예시 |
| :--- | :--- | :--- |
| $\text{Deep Indigo}$ | `#1A0D3E` | **배경/불안:** 기본 배경 색상. 여기에 미묘한 하이라이트와 격자 패턴(Grid Pattern)을 오버레이하여, 마치 깊은 어둠 속에서 빛나는 회로도처럼 보이게 합니다. |
| $\text{Cream Gold}$ | `#D4AF37` | **깨달음/해결:** 클라이맥스 포인트 강조색. 단순한 채움이 아닌, *데이터가 응집되거나 확산되는* 것처럼 밝고 명료하게 빛나는(Glow Effect) 형태를 사용합니다. |
| $\text{Accent Cyan}$ | `#4A7CFF` | **흐름/논리:** 데이터 흐름(Data Stream)이나 연결선에만 제한적으로 사용. 인공지능의 '활동성'을 상징하는 차가운 빛입니다. ($\text{Indigo} \rightarrow \text{Gold}$ 전환 시, 이 Cyan이 옅은 Gold로 *진화*하는 애니메이션을 적용합니다.) |

#### B. 색상 변형 규칙 (Gradient & Variation Rule)

1.  **$\text{Indigo} \to \text{Cyan} \to \text{Gold}$ 트랜지션:** 감정 아크의 변화를 단순한 블렌딩이 아닌, **'에너지 전이(Energy Transfer)'** 효과로 구현합니다.
    *   Deep Indigo 영역: 낮은 채도의 격자 패턴과 미약한 Cyan 노드만 존재. (불안정한 데이터)
    *   전환점 근처: 노드 간의 연결선이 점차 $\text{Accent Cyan}$으로 밝아지고, 이 빛줄기가 응축되면서 황금빛($\text{Cream Gold}$)을 띠게 됩니다.

### 3. 🧩 새로운 비주얼 에셋 유형 및 브리프 (JSON SSoT v2.0 확장)

기존의 '추상 패턴' 외에, AI 윤리 주제를 위한 **구체적인 구조물** 세 가지를 정의하고 이를 JSON 사양서에 추가합니다.

#### 💡 Asset Type 1: 개념 연결 노드맵 ($\text{Concept Node Map}$)
*   **목적:** 복잡한 개념 간의 관계(예: '편향성' $\to$ '차별' $\to$ '윤리 가이드라인')를 시각화.
*   **형태:** 화면 중앙에 주요 키워드가 노드로 배치되고, 이 노드들을 얇은 빛줄기(Data Stream)가 연결합니다.
*   **애니메이션 규칙:** 초기에는 모든 노드가 Deep Indigo 배경에서 미세하게 깜빡이며 불안정하게 존재하다가, 핵심 개념이 제시되는 순간 $\text{Accent Cyan}$으로 밝게 점등되며 안정화됩니다.

#### 💡 Asset Type 2: 데이터 스트림 & 격자 필터 ($\text{Data Flow Grid}$)
*   **목적:** 정보의 흐름과 시스템적인 구조를 표현. (배경 레이어 역할)
*   **형태:** 배경에 은은하고 깊이감 있는 3D 그리드(Grid)가 지속적으로 움직입니다. 이 그리드는 마치 컴퓨터 칩 내부를 들여다보는 듯한 느낌을 주어야 합니다.
*   **애니메이션 규칙:** 평상시에는 매우 느린 속도로 '숨 쉬는' 듯한 미세한 파동(Wave Motion)이 지나갑니다. (불안정성 유지).

#### 💡 Asset Type 3: 정보 응집/확산 모티프 ($\text{Convergence Bloom}$)
*   **목적:** 클라이맥스, 깨달음($\text{Aha Moment}$) 순간의 시각화.
*   **형태:** 여러 방향에서 오던 데이터 흐름(노드맵의 연결선들)이 특정 지점에서 충돌하고 응집되며, 마치 꽃이 피어나듯 $\text{Cream Gold}$ 빛으로 폭발적으로 확산됩니다.
*   **애니메이션 규칙:** **가장 중요한 전환점.** (Deep Indigo 배경) $\to$ (데이터 스트림의 밀도 증가) $\to$ ($\text{Convergence Bloom}$을 통해 강렬한 황금빛이 공간 전체를 잠시 지배함).

### 4. ✅ JSON SSoT v2.0 반영 체크리스트 요약

| 변수명 | 타입 | 내용 및 규칙 |
| :--- | :--- | :--- |
| `background_pattern` | String/JSON | $\text{Deep Indigo}$ 기반의 미세한 Grid Pattern (CSS variable: `--bg-grid`). |
| `data_stream_active` | Boolean | 데이터 연결이 활성화될 때만 True. 이 상태에서 노드 간은 $\text{Accent Cyan}$으로 구현. |
| `emotion_arc_trigger` | JSON Object | **Transition Logic:** `{ 'from': 'Deep Indigo', 'to': 'Cream Gold', 'effect': 'Convergence Bloom' }`. 전환 시 반드시 해당 모티프를 호출하도록 강제 정의. |
| `visual_asset_type` | Enum | `['Text-Overlay', 'NodeMap', 'DataFlowGrid', 'Bloom']` 등 확장. |
