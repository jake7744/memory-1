## 2️⃣ 모션 그래픽 컴포넌트 (Motion Assets & API Spec)

### A. 전환 효과: Aura Dissolve (필수 기본 원칙)
*   **기능:** 한 상태(State A)에서 다른 상태(State B)로 전환될 때 발생하는 시각적 흐름 표현.
*   **API 호출 파라미터:**
    1.  `[Start_Element]` (변화가 시작되는 요소의 위치/좌표): 필수 입력.
    2.  `[End_Element]` (최종적으로 나타날 요소의 위치/좌표): 필수 입력.
    3.  `[Duration_ms]` (애니메이션 지속 시간): 400ms ~ 800ms 권장.
    4.  `[Intensity]` (광원 강도, 0~100%): 전환 시점에 따라 조절.
*   **기술 사양:** 화면 중앙 또는 핵심 요소 주변에서 시작된 빛이 방사형(Radial)으로 퍼지면서 배경의 Deep Indigo 색상과 상호작용하며 사라지고(Dissolve), 다음 컴포넌트가 그 자리를 채우는 방식.

### B. 정보 강조 박스 (Info Box / Concept Container)
*   **목적:** 핵심 개념이나 정의를 잠시 멈추고 시청자에게 깊이 각인시키는 기능.
*   **API 호출 파라미터:**
    1.  `[Content_Text]` (입력할 텍스트): 필수 입력.
    2.  `[Emphasis_Level]` (강조 수준: Low/Medium/High): 강조 강도에 따른 프레임 크기 및 골드 악센트 사용 여부 결정.
    3.  `[Duration_sec]` (표시 시간): 3초 ~ 5초 권장.
*   **디자인 규격:** 배경은 Cream White로 설정하며, 상하좌우에 `Secondary Gold (#D4AF37)`의 얇은 프레임이 적용된다.

### C. 개념 연결 노드맵 (Node Map Visualization)
*   **목적:** 복잡한 무의식/심리 구조 간의 관계(Relationship)를 시각화.
*   **API 호출 파라미터:**
    1.  `[Source_Node]` (출발 노드의 이름 및 좌표): 필수 입력.
    2.  `[Target_Node]` (도착 노드의 이름 및 좌표): 필수 입력.
    3.  `[Relationship_Type]` (관계의 종류: Cause/Influence/Conflict 등): 관계에 따른 선의 스타일 결정.
    4.  `[Activation_Sequence]` (활성화 타이밍): 애니메이션이 순차적으로 일어나야 할 시점 코드(예: T+1.2s).
*   **기술 사양:** 모든 노드는 `Deep Indigo` 배경 위에 Cream White로 표시되며, 연결선은 두 개의 점으로 이루어진 **Golden Accent (`#FFC700`)**의 빛줄기로 시작하여 목표 지점에 도달하며 사라지는 애니메이션을 사용한다.

## 3️⃣ 필수 전송 파라미터 목록 (Essential Transmission Parameters)
모든 에이전트가 콘텐츠를 생성할 때, 아래의 메타 데이터를 반드시 포함하고 API 호출 시 참조해야 합니다. 이들은 컴포넌트를 구성하는 **최소 데이터 세트**입니다.

| 파라미터 이름 | 타입 | 설명 | 필수 여부 | 예시 값 |
| :--- | :--- | :--- | :--- | :--- |
| `[Content_ID]` | String | 현재 콘텐츠의 고유 식별자 (재활용 및 추적). | ✅ 필수 | V4.0-EgoTrip-S2 |
| `[Scene_Timecode]` | TimeCode | 해당 컴포넌트가 화면에 노출되어야 하는 정확한 시간 코드 (예: 00:15:30~00:17:00). | ✅ 필수 | T+1분 5초 ~ T+2분 7초 |
| `[Visual_State]` | Enum | 현재 화면의 감정적/정보적 상태 (Indigo, Transition, Golden). | ✅ 필수 | Transition (불안 $\rightarrow$ 깨달음) |
| `[Text_Input]` | String | 해당 시점에 표시되어야 할 텍스트 데이터. | ✅ 필수 | "무의식은 우리 삶을 지배한다" |
| `[Focus_Keyword]` | Array<String> | 이 섹션에서 가장 강조되어야 하는 핵심 키워드 목록. (골드 악센트 적용 대상). | ✅ 필수 | ["자기 인식", "잠재력"] |
| `[CTA_Trigger]` | Boolean | 해당 컴포넌트가 CTA(행동 유도)를 촉발시키는 지점인지 여부. | ⚠️ 중요 | True / False |
