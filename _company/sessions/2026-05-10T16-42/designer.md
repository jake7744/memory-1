## 🎬 섹션별 시각화 디자인 브리프 (Scene Breakdown)
스크립트는 감정 아크에 따라 최소 3개의 모듈(Module A/B/C)로 분할되어야 합니다. 각 모듈은 아래의 기술 사양을 반드시 준수합니다.

### MODULE A: 불안과 공허함 (Deep Indigo Phase)
*   **감정 상태:** 혼란, 의문, 답답함, 미지의 영역에 대한 두려움.
*   **목적:** 시청자에게 깊은 공감(Empathy)을 유도하여 '나만 이런 게 아니구나'라는 심리적 동조를 일으킨다. (후크 단계)

| 스크립트 내용 예시 | 배경 레퍼런스 타입 (Visual Reference Type) | 타이포그래피 변화 지점 (Typography Change) | 전환 효과 및 애니메이션 스펙 (Transition Spec) |
| :--- | :--- | :--- | :--- |
| *"혹시 나만 이런 감정을 느끼는 걸까?"* (문제 제기) | **[Cinematic Still]** 빗방울이 창문에 부딪히며 흐릿해지는 장면. 초점이 중앙의 공허한 사물(빈 의자, 어두운 방구석)에 맞추어짐. 필름 그레인 강하게 적용. | 모든 문장은 **Deep Indigo**로 표시하며, 크기는 작고 빽빽하게 배치하여 답답함을 시각화. | **[Transition: Slow Dissolve]** 이전 장면에서 현재 화면으로 넘어갈 때, 배경에 미세한 노이즈(Noise)와 함께 천천히 사라지며 다음 문장으로 진입. (속도: 2초) |
| *핵심 키워드:* 불안, 무의식, 질문 | **[Mood/Tone]** 빈티지 필터 적용된 공허하고 깊은 심해 또는 안개 속 풍경. 채도가 낮고 명암 대비가 강함. | 핵심 키워드가 언급될 때만 **Deep Indigo 배경 위에 Cream Gold 하이라이트**로 순간적으로 강조 (Flash Effect). | - |

### MODULE B: 탐색과 고민 (Transitional Phase)
*   **감정 상태:** 의문점 발견, 정보를 찾으려는 노력, 심층적인 자기 성찰.
*   **목적:** 시청자가 능동적으로 '답'을 찾아가도록 유도하며 몰입도를 최고로 끌어올린다. (정보 제공 단계)

| 스크립트 내용 예시 | 배경 레퍼런스 타입 (Visual Reference Type) | 타이포그래피 변화 지점 (Typography Change) | 전환 효과 및 애니메이션 스펙 (Transition Spec) |
| :--- | :--- | :--- | :--- |
| *'진짜 문제는 어디서 올까?'* (정보 제시 시작) | **[Conceptual Motion Graphic]** 복잡하게 얽힌 실타래나 신경망(Neural Network) 그래픽. 여러 개의 노드가 불규칙적으로 깜빡이며 연결되는 모습. | 본문은 **Neutral Cream**으로 가독성을 높이고, 중요한 학술적 용어는 **Deep Indigo 배경 위에 Gold Underline** 처리로 강조. | **[Transition: Node Connect/Morphing]** 하나의 개념(노드)에서 다음 개념(노드)으로 넘어갈 때, 마치 전선이 연결되거나 형태가 변형되는 듯한 부드러운 모션 그래픽을 사용. (속도: 1.5초) |
| *핵심 키워드:* 원리, 패턴, 심리학적 접근 | **[Mood/Tone]** 고대 도서관의 거대한 책장 속에서 빛줄기가 떨어지는 장면. 지식과 탐구의 느낌을 주는 황금 비율의 구도 활용. | 시각 자료가 등장할 때는 폰트를 잠시 크게 확대했다가 줄여주는 (Scale Up/Down) 애니메이션으로 집중도를 높인다. | - |

### MODULE C: 깨달음과 해결책 (Cream Gold Phase)
*   **감정 상태:** 통찰, 해방감, 명료함, 자신감 회복.
*   **목적:** 시청자에게 최종적인 'Aha Moment'를 선사하고, 다음 행동(CTA)으로 자연스럽게 연결한다.

| 스크립트 내용 예시 | 배경 레퍼런스 타입 (Visual Reference Type) | 타이포그래피 변화 지점 (Typography Change) | 전환 효과 및 애니메이션 스펙 (Transition Spec) |
| :--- | :--- | :--- | :--- |
| *"결국 우리가 놓치고 있던 건, 바로 '나' 자신이었어."* (클라이맥스/깨달음) | **[Natural Light]** 안개가 걷히고 따뜻하고 밝은 햇살이 가득 들어오는 창가. 구도 중앙에 깨끗하게 비추어진 물체(예: 손바닥, 열린 문). | 모든 메시지는 **Cream Gold 색상**으로 변경되며, 글자 크기를 최대화하여 명확한 확신을 전달. (Playfair Display Bold를 활용하여 웅장함을 부여) | **[Transition: Flash/Bloom]** Deep Indigo 배경에서 Cream Gold로 전환되는 순간, 화면 전체가 강렬한 황금빛 플래시(Flash)를 터뜨리듯 밝아졌다가 안정화됨. 이펙트 필수 구현. (속도: 0.5초) |
| *CTA 메시지:* "더 깊은 탐험을 원한다면?" | **[Final Visual]** 배경이 완전히 깨끗하고 여백이 많아짐(Negative Space). 시선이 자연스럽게 하단의 CTA 영역으로 이동하도록 유도하는 구도. | CTA 텍스트는 Gold로, 버튼 애니메이션은 **Gold Glow 효과**를 적용하여 클릭 욕구를 극대화한다. | - |
